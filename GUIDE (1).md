## Step별 상세 UI

### Step 1: 이름 (필수)
```
- 텍스트 입력창
- 자동 포커스
- 미입력 시 빨간 테두리 + "이름을 입력해주세요"
- 다음 버튼 비활성화
```

### Step 2: 생년월일 (필수)
```
- ko: 양력/음력 토글 버튼 + 년/월/일 스크롤 드럼
- en/ja/zh: 버튼 없이 날짜 입력만
- 미입력 시 빨간 테두리 + "생년월일을 입력해주세요"
```

### Step 3: MBTI (선택)
```
- 16개 버튼 4x4 그리드
- 선택 시 색상 강조
- [MBTI를 모릅니다 · 건너뛰기 →] 버튼 크게
- 건너뛰기 시 MBTI 분석 제외 처리
```

### Step 4: 태어난 시간 (선택)
```
- 12가지 시(時) 리스트
- 각 시 옆에 실제 시간대 병기 (예: 자시 23:00~01:00)
- [시간을 모릅니다 · 건너뛰기 →] 버튼 크게
- 건너뛰기 시 6글자 사주 분석
- 💡 "태어난 시간을 알면 사주 분석이 더 정확해요!"
```

### Step 5: 얼굴 사진 (선택)
```
- 카메라 촬영 버튼 + 업로드 버튼 나란히
- 모바일: 전면카메라 자동 (facingMode: "user")
- 분석 정확도 바: 사진 없음 75% / 사진 있음 100%
- 보안 안내:
  "🔒 사진은 분석 즉시 자동 삭제됩니다.
   서버에 저장되지 않습니다."
- 사진 없을 때:
  "사진이 없으셔도 괜찮아요!
   이름·생년월일만으로도 분석이 가능합니다."
- [사진 없이 무료 분석 받기 →] 버튼 가장 크게
- 얼굴 미감지 시:
  "얼굴을 찾을 수 없습니다. 다시 시도해주세요."
```

### 모든 데이터 파일 공통 조건
```
1. 역할 명시: "너는 동양 명리학 전문 AI야"

9. 실생활 예시: 현대인이 공감할 구체적 예시 포함
10. 면책 전제: "참고 자료" 전제하에 작성
```

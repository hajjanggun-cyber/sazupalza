# 포스팅 작성 규칙

- 기준 일시: 2026-03-09
- 목적: 새 포스팅을 만들 때 필요한 작성 기준만 남긴 문서

## 1. 기본 원칙

- 글은 한글과 영어를 함께 만든다.
- 글은 최종적으로 `lib/blog/*-posts.ts`에 들어가는 구조를 기준으로 작성한다.
- 카테고리에 맞는 `slug`, `seoTitle`, `seoTitleEn`, `description`, `descriptionEn`, `keywords`, `keywordsEn`이 같이 있어야 한다.
- 글 주제와 `slug`는 정확히 일치해야 한다.
- 같은 주제를 다른 slug에서 중복 타겟팅하지 않는다.

## 2. 포스팅 1개 필수 구성

1. 한글 본문 `contentKo`
2. 영어 본문 `contentEn`
3. 한글 SEO 제목 `seoTitle`
4. 영어 SEO 제목 `seoTitleEn`
5. 한글 메타 설명 `description`
6. 영어 메타 설명 `descriptionEn`
7. 한글 키워드 `keywords`
8. 영어 키워드 `keywordsEn`
9. 한글 목차 `toc`
10. 영어 목차 `tocEn`
11. 한글 FAQ `faq`
12. 영어 FAQ `faqEn`

## 3. 본문 작성 형식

- 본문은 HTML 형식으로 작성한다.
- 기본 태그:
  - `<section>`
  - `<h2 id="...">`
  - `<h3>`
  - `<p>`
  - 필요시 `<ul><li>`
- `toc`의 `id`와 본문 `<h2 id="...">`는 반드시 일치해야 한다.
- 한글/영문 본문은 같은 주제를 다루되, 직역보다 각 언어에서 자연스럽게 읽히는 형태를 우선한다.

## 4. 분량 기준

- 한글 본문은 얇지 않게 쓴다.
- 단순 요약문보다 설명형 정보글 구조로 쓴다.
- 검색 사용자가 바로 이해할 수 있게 아래 흐름이 보이게 쓴다.
  - 정의
  - 핵심 원리
  - 실제 해석 포인트
  - 실전 팁 또는 주의점
  - 결론

## 5. SEO 규칙

- `seoTitle`은 검색 의도를 바로 보여줘야 한다.
- 과장보다 무엇을 알려주는 글인지가 선명해야 한다.
- 연도형 키워드(`2026`)는 연도 의존 주제에만 사용한다.
- evergreen 글에는 불필요하게 연도를 넣지 않는다.
- 메타 설명은 클릭 유도보다 글 내용 요약을 우선한다.
- 키워드는 제목과 본문 주제를 벗어나지 않게 맞춘다.

## 6. 내부 링크 규칙

- 본문 중간에 관련 글 링크를 넣을 수 있다.
- 링크 문구와 실제 목적지 주제가 일치해야 한다.
- 예:
  - `용신` 글에서 `직업운` 글로 보낼 때는 직업운 문구 사용
  - `음력·양력` 글에서 `용신` 문구를 달아 보내면 안 됨
- `relatedPosts`도 같은 원칙을 따른다.

## 7. CTA 규칙

- 본문 중간 또는 하단에 서비스 유도 문구를 넣을 수 있다.
- 과도한 광고 문구보다 자연스러운 서비스 연결이 좋다.
- 기본 방향:
  - 한글: `/ko`
  - 영어: `/en`

## 8. publishedAt 규칙

- `publishedAt`은 실제 신규 발행이나 실질 수정이 있을 때만 변경한다.
- 내용 수정 없이 날짜만 바꾸지 않는다.

## 9. 글 작성 요청 템플릿

```text
다음 조건으로 [카테고리] 포스팅 1개 작성:

주제:
slug:

필수 포함:
1. 한글 본문 HTML
2. 영어 본문 HTML
3. 한글 SEO 제목
4. 영어 SEO 제목
5. 한글 메타 설명
6. 영어 메타 설명
7. 한글 키워드
8. 영어 키워드
9. 한글 TOC
10. 영어 TOC
11. 한글 FAQ 2~3개
12. 영어 FAQ 2~3개

주의:
- slug와 제목과 본문 주제를 정확히 일치시킬 것
- 기존 글과 키워드가 겹치지 않게 할 것
- 내부 링크 문구와 목적지 주제를 맞출 것
```

## 10. 최종 점검 체크리스트

- slug와 주제가 일치하는가
- 한글/영문 제목이 같은 주제를 가리키는가
- 본문과 메타가 같은 검색 의도를 향하는가
- TOC id와 본문 h2 id가 일치하는가
- relatedPosts와 내부 링크 문구가 정확한가
- 기존 글과 키워드 카니발리제이션이 없는가

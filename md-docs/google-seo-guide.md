# Google SEO 공식 가이드 — Codex 글 작성 전 필수 숙지 문서

작성일: 2026-03-06 18:26 KST
출처: Google Search Central 공식 문서 (4개 페이지 통합 정리)

이 문서는 Codex가 글을 작성하기 전 반드시 읽고 숙지해야 하는 Google 공식 SEO 기준이다.
이 기준을 따르지 않으면 아무리 잘 쓴 글도 검색 상단에 노출되지 않는다.

---

## 1. Google 검색의 작동 구조 — Codex가 이해해야 하는 기본

Google 검색은 3단계로 작동한다.

### 1단계: 크롤링
- Googlebot이 웹을 자동으로 탐색하며 페이지의 텍스트, 이미지를 수집한다.
- 내부 링크와 사이트맵을 통해 새 페이지를 발견한다.
- 중요: robots.txt나 잘못된 설정으로 차단된 리소스는 크롤링이 불가능하다.

### 2단계: 색인 생성
- 수집된 페이지의 제목, 본문, alt 속성, 언어, 국가를 분석해 Google 데이터베이스에 저장한다.
- 다음 경우에는 색인이 거부된다:
  - 콘텐츠 품질이 낮을 때
  - noindex 태그가 있을 때
  - 중요 리소스(CSS 등)가 차단되어 있을 때

### 3단계: 검색결과 게재 (순위 결정)
- 사용자 검색어와의 관련성, 콘텐츠 품질, 위치, 언어, 기기를 종합하여 순위를 결정한다.
- 색인에 포함되어도 관련성 또는 품질이 낮으면 노출되지 않는다.

**Codex에게 주는 의미:**
글을 쓸 때 제목, 소제목, 본문이 검색 의도와 정확히 연결되어야 하며, 콘텐츠 자체의 품질이 노출 여부를 결정한다.

## 2. 유용한 콘텐츠의 기준 — Google이 보상하는 글의 조건

Google의 자동 순위 시스템은 "사람에게 도움을 주기 위해 제작된 유용하고 신뢰할 수 있는 정보"를 우선 표시한다.

### 아래 질문에 모두 "예"여야 한다

**콘텐츠 품질:**
- 이 글에 고유한 정보, 분석, 통찰이 있는가? (다른 곳에서 복사하거나 단순 요약하지 않았는가)
- 주제에 대해 완전하고 본질적인 설명을 제공하는가?
- 독자가 읽고 나서 다시 검색할 필요가 없을 만큼 충분한 답변을 주는가?
- 제목이 내용을 정확하게 반영하는가? (과장되거나 낚시성이 아닌가)
- 뻔하지 않은 유용한 분석이나 흥미로운 정보가 있는가?

**전문성과 신뢰:**
- 저자(또는 페르소나)가 이 주제의 권위자로 인식될 수 있는가?
- 검증 가능한 사실 오류가 없는가?
- 신뢰할 수 있는 방식으로 출처나 배경이 공개되는가?

**사용자 경험:**
- 독자가 만족스러운 경험을 얻는다고 느낄 수 있는가?
- 주제 지식을 실제로 학습할 수 있는가?

---

## 3. E-E-A-T — Google이 콘텐츠를 평가하는 4가지 기준

Google은 모든 콘텐츠를 아래 4가지 관점에서 평가한다.

| 기준 | 의미 | 글에서의 적용 |
|------|------|--------------|
| Experience (경험) | 실제 경험에서 나온 정보인가 | 장소나 현상을 직접 관찰한 것처럼 구체적으로 서술 |
| Expertise (전문성) | 해당 주제에 깊은 지식이 있는가 | 피상적인 묘사 금지. 이유와 맥락을 명확히 설명 |
| Authoritativeness (권위성) | 이 주제의 권위자로 인정받는가 | 구체적 분석과 현장 관찰로 전문성 입증 |
| Trust (신뢰성) | 독자가 신뢰할 수 있는가 | **4가지 중 가장 중요.** 오류 없음, AI 사용 공개, 사실 기반 |

**신뢰성(Trust)이 가장 중요하다.** 나머지 세 가지는 신뢰성을 뒷받침하기 위해 존재한다.

## 4. 검색엔진 중심 콘텐츠 — 절대 해서는 안 되는 것

아래에 해당하면 Google이 페널티를 부과하거나 노출을 차단할 수 있다.
이는 스팸 정책 위반이다.

**즉시 금지:**
- 순위 조작을 목적으로 AI 자동화를 사용해 콘텐츠를 대량 생산하는 것
- 다른 글을 단순 요약하면서 독자적 가치를 추가하지 않는 것
- 특정 단어 수를 채우기 위해 분량만 늘리는 것
- 기존 잠재고객 없이 인기 주제만 노리는 것
- 미확인 정보를 사실인 것처럼 확실하게 제시하는 것
- 내용 변경 없이 페이지 날짜만 변경하는 것
- 메타 키워드를 인위적으로 반복하는 것 (Google은 메타 키워드를 사용하지 않음)
- 콘텐츠를 가치 없이 쌓아서 크롤링 신호를 조작하는 것

**핵심 원칙:**
"검색엔진 방문 유도가 주목적"인 글은 Google이 보상하지 않는다.
"사람들을 돕기 위한 목적"으로 쓴 글이 결과적으로 검색 상단에 오른다.

## 5. 제목과 설명(meta description) 작성 기준

### 제목 (title)
- 60자 이하
- 명확하고 간결하게 페이지 내용을 정확하게 설명한다
- 과장되거나 충격적인 표현 금지
- 페이지마다 고유해야 한다
- 중요 정보(장소명, 주제)를 앞쪽에 배치한다

### 설명 (description)
- 120~155자
- 검색결과에서 독자가 클릭할 이유를 제공한다
- 페이지마다 고유하고 관련성이 높아야 한다
- 내용을 단순 반복하는 것이 아니라, 클릭을 유도하는 한 문장으로 설계한다

---

## 6. 콘텐츠 구조 기준

### 본문 구성
- 단락과 소제목으로 명확하게 분할한다
- 소제목은 실제 내용을 반영해야 한다 (장식용 소제목 금지)
- 리스트는 필요한 경우에만 사용한다. 설명 중심으로 쓰는 것이 원칙이다
- 같은 내용을 반복하는 문장을 제거한다 (padding 금지)

### 인트로
- 이 페이지가 무엇을 다루는지 명확하게 정의한다
- 독자가 이 글을 읽어야 하는 이유를 첫 단락에서 제시한다

### 링크
- 앵커 텍스트는 링크 대상 내용을 자연스럽게 설명해야 한다
- 신뢰할 수 없는 외부 링크에는 nofollow 처리한다

## 7. 한글 글과 영어 글의 검색 의도 분리 — SEO 핵심 원칙

Google은 언어와 국가 신호를 수집하여 각 언어 사용자에게 최적화된 결과를 제공한다.
이는 한글 글과 영어 글이 같은 내용의 번역이어서는 안 된다는 것을 의미한다.

- 한글 글: 한국 사용자의 검색 습관과 의도에 맞게 설계
- 영어 글: 외국인 독자의 검색 의도와 맥락에 맞게 설계
- hreflang 태그: Google에 두 언어 버전이 연결되어 있음을 알린다 (`hreflangSlug` 동일값 사용)

**중요:** hreflang이 올바르게 설정되면 Google은 한글 사용자에게 한글 버전을, 영어 사용자에게 영어 버전을 각각 노출한다. 두 글이 번역 관계라면 이 효과가 반감된다.

---

## 8. AI 생성 콘텐츠 — Google 정책 준수

Google은 AI 생성 콘텐츠 자체를 금지하지 않는다.
단, "AI 생성 등의 자동화를 사용하여 검색 순위를 조작하는 것을 주목적으로 콘텐츠를 생성하는 행동은 스팸 정책 위반"이다.

따라서:
- AI를 사용해도 사람에게 도움이 되는 고품질 콘텐츠라면 문제 없다
- 콘텐츠 품질이 공개 여부보다 중요하다
- frontmatter에 `authorName`, `authorRole`, `aiGenerated` 필드를 사용하지 않는다 (AGENTS.md 기준 우선)

---

## 9. 사이트맵과 색인 요청

- 사이트맵은 Google에 중요한 페이지를 알리는 핵심 수단이다
- 글 발행 후 사이트맵 반영 확인 → Google Search Console 수동 색인 요청이 별도로 필요하다
- 사이트맵 자동 반영과 구글 색인 요청은 같은 일이 아니다

---

## 10. 글 작성 전 최종 체크리스트

아래 항목을 모두 통과해야 발행 가능한 글이다.

- [ ] 이 글은 사람에게 도움을 주기 위해 쓰여졌는가? (검색 순위 조작이 목적이 아닌가)
- [ ] 고유한 정보, 분석, 통찰이 있는가? (단순 요약이나 복사가 아닌가)
- [ ] 독자가 이 글 하나로 검색 의도를 완전히 충족할 수 있는가?
- [ ] 제목이 60자 이하이고 내용을 정확하게 반영하는가?
- [ ] description이 120~155자이고 클릭 이유를 제공하는가?
- [ ] 분량 채우기용 문장(padding)이 없는가?
- [ ] 사실 오류가 없는가?
- [ ] hreflangSlug가 쌍 파일과 동일한가?
- [ ] 특정 아이돌·그룹명·기획사 상표가 없는가?
- [ ] 한글 글과 영어 글이 번역이 아니라 각각의 검색 의도로 독립 설계되었는가?
